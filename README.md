## Deskripsi Proyek
Kel2RumahSakit merupakan sistem **Bed Management Rumah Sakit** berbasis web yang dirancang untuk membantu pengelolaan tempat tidur rawat inap secara terpusat dan terstruktur.  
Sistem ini memungkinkan petugas rumah sakit untuk memantau ketersediaan bed, status penggunaan, serta alur rawat inap pasien secara real-time.

Pengembangan sistem ini ditujukan untuk mendukung proses operasional rumah sakit agar lebih efisien, meminimalkan kesalahan pencatatan, serta menyediakan data yang dapat digunakan dalam penyusunan laporan dan pengambilan keputusan manajemen.

---

## Fitur Utama
* Manajemen data pasien
* Manajemen ruangan dan tempat tidur
* Pengelolaan status bed (Kosong, Terisi, Pembersihan, Siap)
* Pencatatan rawat inap pasien
* Pengaturan hak akses berdasarkan peran pengguna
* Laporan indikator Bed Management (BOR, ALOS, TOI, BTO, dll)
